# Data Availability and Release Policy

Build text policy: `redacted`.

Included files may contain de-identified IDs, labels, final adjudication fields,
exact-consensus subsets, sanitized text, derived tables, figures, OOF predictions,
code, and environment metadata.

The release excludes account handles, profile metadata, DIDs, CIDs, URIs,
personal e-mail addresses, local paths, credentials, private validation keys,
internal annotator comments, deprecated LLM files, and checkpoints.

Removing account metadata does not make exact public text unsearchable. Original
social-media content remains attributable to its authors and is not relicensed by
this repository.
