# Human validation package

The repository's quantitative validation is based on the formal human gold produced by independent annotation, review/calibration, effective agreement, and adjudication of remaining applicability conflicts.

Files:

- `formal_applicability_gold_1497.csv`: final applicability for all items;
- `paired_axis_annotations_733.csv`: paired A1/A2 axis labels, including disagreements;
- `exact_consensus_gold_3class_395.csv`: exact 3-class agreement;
- `exact_consensus_gold_5class_244.csv`: exact 5-class agreement.

The external 50-item 2024 challenge set is a qualitative diagnostic sample and is not the basis of the reported supervised validation.
