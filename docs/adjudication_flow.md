# Annotation Review and Adjudication Flow

```text
independent A1/A2 annotation
→ targeted review and calibration
→ effective annotations
→ effective agreement or adjudication of remaining conflicts
→ final applicability gold
```

If effective A1/A2 applicability labels agree, that value enters the final gold.
Only remaining conflicts require an explicit final decision. Final coverage is
not reported as inter-annotator agreement.
