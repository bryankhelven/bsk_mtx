# Data Collection and Scope

The main corpus contains 1497 Bluesky posts. The unit of
analysis is the visible text of one post. External links and profile context are
not consulted during annotation.

The temporal projection contains 989 records
and 973 unique text hashes. It is an
external projection corpus, not a replacement for the adjudicated main gold.

Applicability is evaluated before acceptance and intensity. Off-topic items,
insufficiently grounded items, and texts without evaluative framing about AI in
education are excluded.
